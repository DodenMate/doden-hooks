# 快速上手

dodenHooks 是一个 React Hooks 库，致力于提供常用且高质量的 Hooks。

>

Demo:

```tsx
import React from 'react';
import { Foo } from 'doden-hooks';

export default () => <Foo title="First Demo" />;
```

More skills for writing demo: https://d.umijs.org/guide/demo-principle
